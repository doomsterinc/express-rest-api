> Node.js REST API Example with MongoDB, Mongoskin, Express 3 and 4


# Express.js 3.x


Instructions:

```
$ npm install
$ node express.js
```

in a new window

```
$ mocha express.test.js
```

or (if you don't have mocha installed globally):

```
$ ./node_modules/mocha/bin/mocha express.test.js
```
